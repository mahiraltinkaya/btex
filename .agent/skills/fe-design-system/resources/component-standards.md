# Component Standards

## UI stack
- Shadcn/ui as the base.
- Tailwind utility-first.
- Icons: Lucide React.

## Section layout
- Use `SectionLayout` for page sections.
- Default spacing: `py-12 md:py-16`.
- Use `border-b` with `border-primary/5` between sections.

## Cards
- Cards use `rounded-xl` or `rounded-2xl`.
- Use `bg-white` with subtle border and `shadow-sm` or `shadow-lg`.

## Accent usage
- Use palette accents as soft backgrounds or borders only.
- Large solid accent blocks are not allowed.

## Accessibility
- WCAG AA contrast.
- Visible focus states.
- Keyboard navigation.
- ARIA labels where needed.
