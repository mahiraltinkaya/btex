# Layout Patterns

## Page shell
- Root: `bg-background/50` with `min-h-screen`.
- Decorative blobs: use soft `bg-primary/5` and blur utilities.

## Hero section
- Container: `container mx-auto w-full px-4 xl:px-32`.
- Grid: `grid grid-cols-1 lg:grid-cols-2` with `gap-12`.
- Headline: `text-4xl lg:text-6xl` with `tracking-tight`.

## Section separators
- Each section uses `border-b border-primary/5`.
- Alternate background using `bg-background/50` or `bg-white/50`.
