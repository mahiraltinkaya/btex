# Design Review Checklist

- Primary color used correctly.
- Background color applied.
- Glassmorphism only on overlays.
- Spacing uses 4px scale.
- Typography rules followed.
- Minimalist layout maintained.
- Shadcn/ui components reused.
- SectionLayout used for sections.
- Section separators use `border-primary/5`.
- Accessibility checks passed.
