# Glassmorphism Rules

## Allowed (overlay only)
- Dialog, Modal, Popover, Tooltip, DropdownMenu, Select, Sheet, Command.

## Not allowed (static)
- Main layout, cards, inputs, navigation bar, footer.

## Tailwind pattern
`bg-white/70 backdrop-blur-md border border-primary/10 shadow-2xl shadow-primary/8`
